# Sticky Notes

A small, joyful sticky-notes board that lives entirely in your browser. Notes
persist via `localStorage` — no backend, no accounts.

Built with Vite + React 18 + TypeScript + Tailwind CSS. Deployed via Vercel
from this GitHub repo.

## Develop

```bash
npm install
npm run dev
```

The dev server runs on http://localhost:3000.

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushing to `main` triggers an automatic Vercel deploy.
